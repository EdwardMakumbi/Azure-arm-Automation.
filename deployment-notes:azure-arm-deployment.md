# Deployment Notes – Azure ARM

### Step 1: Resource Group
- Create Resource Group in Azure Portal

### Step 2: Template Deployment
- Upload ARM template JSON file
- Validate and deploy template
- Ensure VNet, Subnets, Storage Accounts, VMs created

### Step 3: VM Configuration
- Configure network access, internal connectivity
- Install required software on VMs

### Step 4: RBAC
- Onboard users to Azure AD
- Assign custom roles per principle of least privilege

### Step 5: Testing
- Validate connectivity between VMs and storage
- Test internal network isolation and access permissions
