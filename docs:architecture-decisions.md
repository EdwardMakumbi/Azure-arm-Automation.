# Architecture Decisions – Azure ARM Template

## IaC & Automation
- ARM templates chosen for repeatable deployments
- Templates reusable across environments

## Security
- Network isolation between VMs and Storage Accounts
- RBAC for users: least privilege enforced

## Resource Organization
- Resource groups for environment separation
- Naming conventions and tagging for governance

## Operational Efficiency
- Scripts reduce manual deployment errors
- Monitoring enabled via Azure Portal
